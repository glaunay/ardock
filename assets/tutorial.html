What is the ARbitrary DOCKing server?

ARDOCK is a web-service for the study of protein surfaces. Arbitrary docking can reveal potential interaction sites on the surface of a protein using docking with a set of 25 random protein “probes”. We have shown that the random probes interact in a non-random manner on protein surfaces, and that targeted regions are enriched in biological interfaces.

ARDOCK uses PDB structures as inputs. The web-client provides the interactive vizualisation of protein structures and sequences. The back-end server performs the protein surface study computations. ARDOCK uses the webGL NGL library for molecular rendering and the HeX software for fast docking computations.

Submitting structures

Any file following the PDB standard can be processed. In its current state, the ARDOCK server only treat natural amino-acids, any other compound will be discarded.

Working with proteins

For multimeric proteins, each oligomeric state will give different results because the ARDOCK method is sensitive to the overall shape of the protein. If the protein you are working on contains several chains, any oligomeric state can be constructed by selectively deleting chains. If you press the + tab button, a copy of the current structure will be opend in a new tab, allowing you to work on a different oligomeric state.
